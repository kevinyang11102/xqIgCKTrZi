# 海鲜自助餐厅管理系统 java519

## 项目概述

本项目是一个基于SSM（Spring + SpringMvc + Mybatis）和Vue的海鲜自助餐厅管理系统。采用经典的三层MVC体系结构，实现了前后端分离的设计模式。

## 技术栈

- 前端：Vue
- 后端：JavaEE、Spring、SpringMvc、Mybatis
- 数据库：MySQL
- 项目构建：Maven

## 功能模块

系统主要包含以下功能模块：

1. 用户管理
2. 订单管理
3. 菜品管理
4. 库存管理
5. 营业统计

## 系统角色

- 管理员
- 服务员
- 厨师

## 运行环境

- 操作系统：Windows/Linux/MacOS
- 数据库：MySQL 5.7+
- 服务器：Tomcat 8.5+
- JDK版本：1.8+

## 部署步骤

1. 导入MySQL数据库文件
2. 部署后端代码到Tomcat服务器
3. 部署前端代码到Web服务器
